Project 2

So this time around we are making a spaceship interface. This is our first project that makes use of the Arduino IDE so I look forward to playing around with that but first things first I need to build my circuit. I started by laying out all of the parts that I needed. Getting everything together was a lot easier this time because of the organizer box that I got.

After that, I hooked up all of the LEDs with 220k ohm resistors and fed them power from the 3, 4, and 5 pins on the microcontroller.

I saved pin 2 for the input from the button which I added next. I used a 110k ohm for this one.

Once all of the elements were connected I added the decoration that was provided in the box. I think it makes the whole this come together and is pretty fun.

Hopping over to the code it was very simple to follow along with the book and write the program to control the LEDs with the button.

Finally, I had everything working here it is in action!
