Project 7

In Project 7 we will be making a little keyboard with which we can press different buttons to play different notes using the piezo. As always I started by laying out all of my components before making my circuit. This time we are using a lot of buttons in a ladder which is cool.

The first bit was easy as it was the exact same setup for the piezo as in Project 6.

After that, I got the ground side of my buttons all wired up.

The final step which was a bit tricky was getting all of my resistors in the right spot. I needed to make sure that I always knew which resistor was which because I couldn’t tell the difference between them on my own and my roommate was away while I was doing this. In the end, I got everything in the right spot.

The code on this project was very simple and didn’t have any code I hadn’t seen before. Just a bit tedious getting the if-else statements set up.
