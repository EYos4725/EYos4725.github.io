Project 10

Still waiting for my replacement board to arrive any day now. This is a shame because this project looked like it would be really fun to see running. Today we will be building a Zoetrope which is like an old false motion machine from the beginning of animation. Unfortunately, I still have to build the circuit without seeing it in action. Nevertheless, I start this project as with all projects by getting all of my components together.

The first step was getting the buttons set up. This was easy enough as I have done all of this before.

The second step was hooking up the potentiometer for speed control. Again this was a breeze to complete which led to the next step which was a lot more complicated.

This final step of completing the circuit was much more daunting than the last two as it required the use of a new component called an H-Bridge. This step was all about making sure that the board was the right way up and that all of the pins went to the right places. In hindsight, if I were to do this again I would have used different colored wires to connect each pin so it was easier to keep track. Regardless, the last step was to hook up the DC motor and 9V battery.
