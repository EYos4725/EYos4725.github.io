Project 8

In Project 8 we will be making a digital hourglass using the new tilt sensor. As you know the drill by now I start the project by gathering my materials and laying them all out.

This time instead of lots of buttons we will be using lots of LEDs. They were all connected in the same way so it was very easy to just go down the line and hook all of them up.

After that, it was a simple task to wire up the tilt sensor and we were good to go for the circuit.

Code this was a breeze. I have gotten used to not having IntelliSense by now so I made very few errors when I was typing it out.
