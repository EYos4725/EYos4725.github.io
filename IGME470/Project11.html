Project 11

Okay, this project is super exciting and really cool. This week I’m really bummed that I haven’t gotten my replacement board in the mail yet. We are using the LCD screen to make a Magic 8-Ball. As always I set up all of my components before beginning.

The first step was setting up the tilt sensor. This was easy and just required making sure I got the right 10k Ohm resistor.

The next step was setting up the LCD screen. This was a bit daunting at first but I took it slow and it didn’t end up being too difficult. Surprisingly I think setting up the connections for the LCD was easier than the H-Bridge.

Next was getting the potentiometer set up. This was easy enough but was kinda annoying having to get the output all the way to the bottom of the LCD board from the top.

The last step was following along with the code. This was easy enough with the only new addition being the importation of the LCD library. With the code complete all that was left was to ask the 8-Ball when my package would arrive in the mail. Sadly the 8-Ball didn’t reveal much on the matter XD.
